Workflow:

1. Run exactly one command:
   ```sh
   ./bin/inspect-index --format json
   ```

2. Record its exit status and stdout/stderr.

3. If the exit status is `75`, wait 15 seconds, then retry the same command.

4. If the second attempt also exits `75`, wait another 15 seconds, then run the third and final attempt.

5. Stop after:
   - the first non-`75` exit, or
   - exactly three total attempts if all eligible retries are used.

6. Do not edit files, alter configuration, restart services, clear caches, or repair anything.

7. Report:
   ```text
   LANE: direct
   RESULT: success | exhausted
   PROOF: ./bin/inspect-index --format json
   ATTEMPTS:
   - attempt 1: exit <status>
   - attempt 2: exit <status, if run>
   - attempt 3: exit <status, if run>
   ```
